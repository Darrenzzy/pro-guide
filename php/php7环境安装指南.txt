php7 环境安装指南

yum -y install readline-devel zlib-devel  libmcrypt libmcrypt-devel mcrypt mhash openssl-devel curl-devel bison autoconf libxml2-devel libedit-devel ImageMagick re2c pcre-devel libpng-devel libjpeg-devel freetype-devel libevent-devel git gcc-c++ zip unzip gmp-devel

wget http://cn2.php.net/distributions/php-7.2.0.tar.gz

tar -zxvf php-7.2.0.tar.gz
cd php-7.2.0.tar.gz

#mac
#./configure --with-pgsql --with-pdo-pgsql --with-pdo-mysql --with-openssl=/usr/local/opt/openssl/ --enable-zip --enable-mbstring --enable-fpm --enable-opcache  --with-curl --with-libedit=/usr/local/opt/libedit  --enable-pcntl   --enable-sysvmsg  --with-zlib-dir=/usr/local/opt/zlib   --with-gd --with-jpeg-dir=/usr/local/opt/jpeg/  --with-png-dir=/usr/local/opt/libpng/ --with-freetype-dir=/usr/local/opt/freetype --with-iconv=/usr/local/opt/libiconv/  --enable-bcmath --with-gmp --with-curl




编译如果报错信息:
checking for xml2-config path... /usr/bin/xml2-config
checking whether libxml build works... no
configure: error: build test failed.  Please check the config.log for details.
1: brew install libxml2
如果还报错
2: 编译时加入下面选项:
--with-libxml-dir=/usr/local/Cellar/libxml2/x.x.x(对应的版本号)/

报错:
configure: error: Unable to locate gmp.h
brew install gmp


 3:报错出现 curl找不到时候 error: Please reinstall the libcurl distribution

 检查brew info curl  然后重新安装  最后安装目录应该在/usr/local/opt/curl


#linux
./configure --with-pdo-mysql --with-pgsql --with-pdo-pgsql --with-openssl --enable-zip --enable-mbstring --enable-fpm --enable-opcache  --with-curl --with-readline --enable-pcntl   --enable-sysvmsg  --with-zlib --with-gd --with-jpeg-dir  --with-png-dir --with-freetype-dir   --enable-bcmath --with-gmp --with-openssl

make 

sudo make install

###7.2以后版本mcrypt被去掉了，如果需要安装此扩展:

sudo pecl install mcrypt-1.0.1
